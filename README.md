# CosmoWiki: Interacciones y Fusiones de Galaxias 🌌

Este sitio web es un artículo de divulgación científica simplificado diseñado con el estilo visual clásico de Wikipedia.

## 🎓 Propósito del Proyecto
Este proyecto fue realizado especialmente como trabajo práctico para el **Diplomado de Honor en Astronomía** (UDP, Edición 2026).

## 🚀 Enlace al Sitio Web
Puedes visitar la página en vivo a través del siguiente enlace:

👉 **[Visitar CosmoWiki en vivo](https://diegov73.github.io/wikiAstronomia/)**

### 🛠️ Comandos útiles para desarrollo local:
* **Instalar dependencias**: `npm install`
* **Servidor local con recarga automática**: `npm run dev`
* **Despliegue automático**: Realiza un `git push` a tu repositorio en GitHub.
